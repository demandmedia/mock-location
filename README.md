----
- Mocha

### npm scripts

#### `npm test`
Runs `mocha` tests located in `/test` once.
#### `npm run cover`
Generates a code coverage HTML report in `/coverage`.
#### `npm run tdd`
Runs `mocha` tests with a minimal reporter.
#### `npm run coverage`
Launches a server and opens the coverage report in your default browser
